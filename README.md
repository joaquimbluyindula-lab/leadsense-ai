# 🎯 LeadSense AI — Triagem Inteligente de Leads

Interface visual para classificação automática de leads usando IA.

## Como funciona
- Lead envia mensagem (WhatsApp, formulário, e-mail)
- n8n recebe via Webhook e envia para Groq LLaMA 3.3
- IA classifica como QUENTE / MORNO / FRIO
- Interface exibe resultado com prioridade e ação recomendada

## Tecnologias
- n8n (automação e webhook)
- Groq API + LLaMA 3.3 70B
- HTML, CSS, JavaScript puro
